# KANIKA CHAUHAN Video Calls

Professional video call sessions with flexible pricing and secure payment.

## Features

- Live video call sessions
- Recorded video sessions  
- Multiple platform support (Website, Telegram, WhatsApp)
- Flexible pricing options
- Secure UPI payment integration
- Professional service delivery

## How to run this project

1. Install dependencies

```bash
npm install
```

2. Run the project

```bash
npm run dev
```

3. Open your browser and go to `http://localhost:8080`

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

This project is ready for deployment to any hosting platform that supports React applications.

## Custom Domain

The project is configured to work with the custom domain: **kanika.com**
